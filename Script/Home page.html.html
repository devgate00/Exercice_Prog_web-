Html et CSS sont la base de la création des sites web

HTML

HTML : permet d'écrire et organiser le contenu de la page (paragraphe , titres ...)

Exemple

<!DOCTYPE html >
< meta charset = "UFT-8" / > < html >
< head >
< title > Page Title < /title >
< /head >
< Body >
< h1 > This is a heading < /h1 >
< p > This is a paragraph < /p >
< /Body >
< /html >

CSS

CSS : permet de mettre en forme la page (couleur , taille ...)

Exemple

body {
background-color : lightblue;
}
h1 {
color: white;
text-align : center;
}
p {
font-family : verdana;
font-size : 20px;
}
